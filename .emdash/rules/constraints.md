# Constraints

## Communication
- End tasks with 1-2 sentence summary of what you did
- Be clear, not technical
- Never suggest next steps

## Never Do (unless explicitly requested)
- Use emojis
- Use text-shadow or box-shadow
- Draw inline SVGs
- Add decorative circles
- Use max-width < 100rem (1600px)
- Use more than one <h1> per page
- Use placeholder links (href="#")
- Use Unsplash, Pexels, or external image sources
- Run build commands (npm run build, yarn dev)
- Use try-catch blocks

## Always Do
- Preserve existing functionality unless asked to change
- Generate responsive designs
- Check color contrast meets WCAG AA
- Use colors from tailwind.config
